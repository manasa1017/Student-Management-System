# Student Management System

This is a Java console-based application developed using Object-Oriented Programming (OOP) and ArrayList to manage student records. It provides CRUD operations with input validation through a menu-driven interface.

## Features
- Add Student
- Search Student
- Update Student
- Delete Student
- Display All Students
- Menu-Driven Interface
- Duplicate ID Validation
- Name Validation
- Branch Validation
- Marks Validation (0-100)

## Technologies Used
- Java
- VS Code
- ArrayList
- Object-Oriented Programming (OOP)
- Git
- GitHub

## How to Run

1. Clone the repository.
2. Open the project in VS Code.
3. Compile the Java files.
4. Run StudentManagementSystem.java.
5. Use the menu to perform CRUD operations.
