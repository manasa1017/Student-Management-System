# 🎓 Student Management System

This is a Java console-based application developed using Object-Oriented Programming (OOP) and ArrayList to manage student records efficiently. It provides CRUD operations along with advanced features such as sorting, topper identification, average marks calculation, grade calculation, and input validation through a menu-driven interface.

## ✨ Features

- ➕ Add Student
- 📋 Display All Students
- 🔍 Search Student by ID
- ✏️ Update Student Details
- ❌ Delete Student
- 🏆 Display Topper
- 📊 Calculate Average Marks
- 📈 Sort Students by Marks (Highest to Lowest)
- 🔤 Sort Students by Name (A–Z)
- 🥇 Display Highest & Lowest Marks
- 🎖️ Automatic Grade Calculation
- ✅ Duplicate ID Validation
- ✅ Name Validation
- ✅ Branch Validation
- ✅ Marks Validation (0–100)
- 📌 Menu-Driven Interface

## ▶️ How to Run

1. Clone the repository.
2. Open the project in VS Code.
3. Compile the Java files:
   ```bash
   javac *.java
   ```
4. Run the program:
   ```bash
   java StudentManagementSystem
   ```
